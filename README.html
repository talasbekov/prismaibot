// tech stack

Полный стек технологий

⚡ Backend

FastAPI
Быстрее Django в 2–5x. Встроенный OpenAPI/Swagger. Async из коробки. Автодокументация.
SQLModel
Создан автором FastAPI. Объединяет SQLAlchemy + Pydantic — нет дублирования кода моделей.
PostgreSQL
Full ACID, JSON, отличная масштабируемость. Самая надёжная production СУБД.
Alembic
Де-факто стандарт для миграций SQLAlchemy. Автогенерация из моделей.
PyJWT + OAuth2
Stateless аутентификация. JWT с 8-дневным TTL. Полная совместимость с OAuth2.
Argon2 (pwdlib)
Победитель Password Hashing Competition 2015. Защищён от GPU-атак. Лучше bcrypt.
Pydantic v2
Написан на Rust. В 5–50x быстрее Pydantic v1. Строгая валидация данных.
uv
Python package manager на Rust. В 10–100x быстрее pip. От авторов ruff.
Ruff + mypy
Ruff заменяет flake8+isort+black. mypy в строгом режиме — полная типобезопасность.
Sentry SDK
Production мониторинг ошибок. Трекинг производительности.

🎨 Frontend

React 19
Новый компилятор, лучший concurrent rendering. Самая популярная UI-библиотека.
TypeScript 5.9
Статическая типизация = меньше ошибок в runtime. Идеально с auto-generated клиентом.
Vite
Старт dev-сервера <300ms. Мгновенный HMR. В 10–20x быстрее webpack.
TanStack Router
File-based routing с полной типобезопасностью. Автоматический code-splitting.
TanStack Query
Умный кеш. Автоинвалидация. Loading/error states. Optimistic updates из коробки.
shadcn/ui
Компоненты которыми ты владеешь. Radix UI (доступность) + Tailwind.
Tailwind CSS 4
Нативные CSS cascade layers. Нет runtime. Utility-first подход.
Zod + RHF
TypeScript-first валидация. Перформанс-ориентированные формы без лишних ре-рендеров.
Biome
Rust-based замена ESLint + Prettier. В 15–25x быстрее.
Playwright
E2E тесты от Microsoft. Кросс-браузерный. Параллельный запуск.
Bun
JavaScript runtime на Zig. В 3–5x быстрее npm/yarn.
@hey-api/openapi-ts
Автогенерация TypeScript клиента из OpenAPI схемы. Ноль ручных API-запросов.

🚀 DevOps

Docker + Compose
Изолированные окружения. Воспроизводимые сборки. Dev/prod паритет.
Traefik 3
Автоматический SSL от Let's Encrypt. Динамический конфиг. Docker-native.
GitHub Actions
CI/CD прямо в репозитории. Параллельный запуск тестов. Deploy в staging/production.
Adminer
Лёгкий веб-интерфейс для PostgreSQL. Один контейнер, удобная инспекция БД.
Mailcatcher
Перехватывает все email в dev-режиме. Веб-интерфейс для просмотра писем.

Клонирование и настройка
bash
cd /home/erda/Загрузки/full-stack-fastapi-template-master
cat .env
Открой .env и измени критичные значения:
.env
# ОБЯЗАТЕЛЬНО измени:
SECRET_KEY=<сгенерируй_случайную_строку>
FIRST_SUPERUSER=admin@yourdomain.com
FIRST_SUPERUSER_PASSWORD=<надёжный_пароль>
POSTGRES_PASSWORD=<надёжный_пароль>

python3 -c "import secrets; print(secrets.token_urlsafe(32))"

04
Локальный запуск без Docker

Backend (Python 3.10+)

bash
curl -LsSf https://astral.sh/uv/install.sh | sh
cd backend && uv sync
docker compose up db -d          # только PostgreSQL
uv run alembic upgrade head
uv run python app/initial_data.py
uv run fastapi dev app/main.py

Frontend (Bun)

bash
curl -fsSL https://bun.sh/install | bash
cd frontend && bun install && bun dev
